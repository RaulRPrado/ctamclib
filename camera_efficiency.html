Camera Efficiency¶

Main functionality for the camera_efficiency module is to provide an interface to the testeff program provided by sim_telarray. A Telescope Model (see telescope_model) and few more physical parameters (zenith angle, etc) are required inputs.

Analyze camera efficiency output file and store the results in _results.
