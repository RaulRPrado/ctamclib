Auxiliary FilesÂ¶

Auxiliary parameter files can found in the data directory:

  1. data/parameters: parameter type and unit definitions (e.g., defining the units for zenith angle for configuration parameters for CORSIKA)

  2. data/layout: layout definitions for single telescope or simple 4-telescope grid layouts (used e.g., for trigger simulations)

All data files related to the Monte Carlo are read from the Model Parameters DB.
