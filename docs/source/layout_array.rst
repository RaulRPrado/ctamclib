Layout
======

In this section you find the reference documentation of the Layout Array and  Telescope Data modules.
The Layout Array module deals with the telescope positions and its coordinate transformations.
It is used within Corsika Config and Array Model in order to incapsulate the information about the
telescope positions.

* `layout_array <layoutarraymodule>`_
* `telescope_data <telescopedatamodule>`_
