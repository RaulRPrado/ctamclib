CORSIKA and sim_telarray¶

Support module for running CORSIKA and sim_telarray.

corsika_simtel_runner¶

class corsika_simtel.corsika_simtel_runner.CorsikaSimtelRunner(common_args=None, corsika_args=None, simtel_args=None)[source]¶

CorsikaSimtelRunner is responsible for running CORSIKA and piping it to sim_telarray using the multipipe functionality. CORSIKA is set up using corsika_autoinputs program provided by the sim_telarray package. It creates the multipipe script and sim_telarray command corresponding to the requested configuration.

It uses CorsikaConfig to manage the CORSIKA configuration and SimtelRunnerArray for the sim_telarray configuration. User parameters must be given by the common_args, corsika_args and simtel_args arguments. The corsika_args and simtel_args are explained in CorsikaRunner and SimtelRunnerArray respectively. An example of the common_args is given below.

common_args = {
    'label': 'test-production',
    'simtel_source_path': '/workdir/sim_telarray/',
}
Parameters:
common_args: dict

Arguments common to both CORSIKA and sim_telarray runners

corsika_args: dict

Arguments for the CORSIKA runner (see full list in CorsikaRunner documentation).

simtel_args: dict

Arguments for the sim_telarray runner (see full list in SimtelRunnerArray documentation).
